# 仙台市営バスマップ

仙台市営バスのバスの現在位置を時刻表データに基づいて表示するウェブアプリです。

## 主な機能

- 時刻表に基づくバスの現在位置の表示
- 各バス停の通過時刻リスト（時刻表）の表示
- 路線ごとの走行ルート表示（仙台駅周辺などがちょっと変だけど）
- 停留所ごとの「次のバス」を表示

## 出典・ライセンス

このアプリケーションは、以下のオープンデータを利用しています。

### バス運行データ

- **データ名称**: 仙台市営バス情報（令和7年4月1日改正ダイヤ）
- **提供元**: [仙台市オープンデータカタログページ](https://www.city.sendai.jp/opendata/index.php)
- **ライセンス**: [クリエイティブ・コモンズ 表示 4.0 国際 (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/deed.ja)

### 地図データ

- **地図タイル**: [国土地理院 地理院タイル 淡色地図](https://maps.gsi.go.jp/development/ichiran.html)
- **地図エンジン**: [MapLibre GL JS](https://maplibre.org/)
- **ルート検索**: [OSRM (Open Source Routing Machine)](http://project-osrm.org/)
